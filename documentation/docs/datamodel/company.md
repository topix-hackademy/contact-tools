# Company

In this model we are going to save all the list of companies. For each Company we save this list of attributes:

* `company_custom_id`
* `company_name`
* `company_short_name`
* `company_business_name`
* `company_vat_number`
* `company_tax_code`
* `company_address`
* `company_cap`
* `company_city`
* `company_province`
* `company_country`
* `company_phone_number`
* `company_fax`
* `company_website`
* `company_notes`
* `creation_date`
* `company_type`
